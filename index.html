Github Live Website Tutorial

This is the complete guide to Github live websites

Github has a very handy live website tool, in this tutorial are the steps for using it

  • Step 1:

    Click on New and Select New Repository

  • Step 2:

    Give your new Repository a name and click "initialize this Repository with a README"

  • Step 3:

    Click "Create Repository"

  • Step 4:

    Drag and drop files into Github

  • Step 5:

    Click "Commit directly to the master branch" and then select "Commit changes"

  • Step 6:

    From here click on settings

  • Step 7:

    Scroll all the way down to the bottom

  • Step 8:

    Click source and select "master branch"

  • Step 9:

    Your site should be published, follow the provided link
